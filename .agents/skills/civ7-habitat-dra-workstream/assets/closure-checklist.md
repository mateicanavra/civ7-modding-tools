# Habitat DRA Closure Checklist

```markdown
- [ ] Product outcome protected by this workstream is named.
- [ ] Stage 0 row exists and matches current evidence.
- [ ] Active packet or ledger row tasks are current.
- [ ] Proof classes are labeled and non-claims are explicit.
- [ ] Injected violation or failure reproduction is present where relevant.
- [ ] Safe-write, rollback, and cleanup proof exists for apply paths.
- [ ] Accepted P1/P2 findings are repaired, source-rejected, invalidated by later evidence, or moved outside scope with owner and trigger.
- [ ] Downstream realignment ledger records touched and checked records.
- [ ] Command proof log contains fresh branch/commit/state evidence.
- [ ] Graphite branch/stack state is recorded.
- [ ] Commit exists on the intended stack layer.
- [ ] Worktree is clean.
- [ ] Supervisor accepts closure.
```
