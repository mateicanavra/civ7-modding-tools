# Packets

This directory contains **source packets** (imported proposal docs + reconciliation notes) for the Foundation evolutionary-physics refactor work.

- `foundation-proposals/`: one proposal set imported from an old PR commit.
- `foundation-refactor-proposal-packet/`: another proposal set + supporting specs.
- `realism-packet/`: reconciliation notes and positioning across sets + current code.
- `m11-foundation-realism-spike/`: pointer to the recent Foundation realism spike source set (audit + guardrails).
- `phase2-foundation-vertical-refactor/`: pointer to contract/migration posture docs.
