4. Then handle Grit diagnostic acquisition separately from Pattern Authority and guarded apply.
5. Move or label MapGen-specific apply gates before claiming generic Transformation Transaction boundaries.
6. Treat hooks after check/classify/proof contracts are stable, because hooks consume those surfaces but own only local feedback.
7. Defer Authoring Topology until a product owner accepts a concrete MapGen convention and acceptance loop.

## Stop Conditions

- Stop if a packet cannot name one owner, one consumer, a public/private contract boundary, and the proof class it preserves.
- Stop if a packet is described as "clean up module X" rather than a scenario/authority/proof simplification.
- Stop if the proposed write set touches MapGen authoring topology without an accepted product convention and executable acceptance loop.
- Stop if a simplification removes an explicit refusal, non-claim, baseline contract failure, rollback proof, or selector failure without replacing it with an equally testable contract.
- Stop if two adjacent packets introduce abstractions but do not reduce reachable state, duplicate command paths, or explicit coupling.
