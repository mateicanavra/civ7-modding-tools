# MapGen how-to guides

## Purpose

How-to guides are task-oriented.
They assume you already understand the basics and want to accomplish a specific task quickly.

## Audience

- Developers extending MapGen or integrating it into apps/workers.
- Authors making new knobs/presets (where supported).
