# Phase Record: Habitat Artifact Term Closure

## Objective

Close Habitat generic artifact language while preserving Civ7/MapGen product
artifact language.

## Allowlist

- `artifact:*` dependency tags.
- Recipe artifacts and generated map artifacts.
- `defineArtifact`.
- `deps.artifacts`.
- Product files whose domain is map generation artifact authority.

## Forbidden Habitat Terms

- `artifactKind`
- manifest top-level `"artifacts"`
- `RuleArtifact`
- `artifact-paths`
- `habitatArtifacts`
- `habitat-artifacts`
- `Habitat artifact`
- `authored artifact`
- `pnpm artifacts`

## Status

Opened for implementation on `codex/habitat-artifact-authority-workstream`.
