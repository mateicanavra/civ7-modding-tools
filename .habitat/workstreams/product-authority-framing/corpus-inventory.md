## Source Classification Rules

- Primary authority: user/repo instructions, AGENTS, accepted baselines, canonical docs, ADRs/deferrals.
- Secondary evidence: source and tests for constructibility/current behavior.
- Discovery material: archives, old issue docs, project scratch, generated summaries.
- Excluded from authority: generated output as editable source, ungrounded current folder shape, package tests used as stale-policy junk drawers.
