# Docs Site Sidebar Operation

Operation: `generate`

This subject is a Habitat-owned docs generation operation, not a registered
read-only check. The current Toolkit registry has no typed operation manifest
schema for generate operations, so this Markdown file is the provisional
identity record.

The operation refreshes `docs/_sidebar.md` from the current docs tree for the
local docs site preview workflow.
