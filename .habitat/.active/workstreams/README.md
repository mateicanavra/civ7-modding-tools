Consolidation initiative. The initiative is the umbrella objective, not a
container directory. Each direct child is a workstream or a workstream index.

Apply the sole canonical source order in `.habitat/.active/frames/FRAME.md`.
This index routes active workstreams only and cannot reorder that authority.

## Operating Rule

Workstream material for this initiative lands as a direct child of this
directory and uses verb-first names that state the work being done. New method
frames land in `.habitat/.active/frames/`. Domino sequence material lands in
`.habitat/.active/dominoes/`, because dominoes are higher-level than any single
workstream.
