# Habitat Active Work

Status: active working-material index

`.habitat/.active/` holds active Habitat workstream material that is still
being shaped, reviewed, or executed. It is the home for Habitat method frames,
workstream ledgers, receipts, decision packets, scopes, slices, investigation
artifacts, and steward scratchpads.

Canonical root authority remains at `.habitat/` for ontology, tree shape,
blueprints, niche rule packets, registry metadata, subject categories,
operation kinds, and tool-separation authority.

Current active lanes:

- `frames/`: all Habitat method frames.
- `dominoes/`: active Habitat Authority Consolidation domino sequence, with
  one file per domino and a complete index.
- `workstreams/`: active Habitat Authority Consolidation workstreams, kept as
  sibling workstreams rather than nested under an initiative wrapper.
